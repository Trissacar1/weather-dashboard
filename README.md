# Weather Dashboard

A browser-based weather web application that displays real-time weather
conditions for multiple locations simultaneously.

## Features
- Fetches live weather data from a public weather API
- Displays current temperature and wind speed
- Built with vanilla HTML, CSS, and JavaScript
- No backend or API keys required

## Tech Stack
- HTML
- CSS
- JavaScript
- Open-Meteo API
- GitHub Pages for hosting
